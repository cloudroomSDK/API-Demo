# 环境要求

|平台|系统要求|工具要求|
|---|---|---|
| Windows | Vista+ | Unity 2021.3.33f1 |


# 运行Demo操作步骤

* 下载SDK

|平台|SDK包|
|---|---|
| Windows | [下载SDK](https://www.cloudroom.com/api/getDownloadUrlApi?Client=windows_unity_sdk) |

* 将sdk包中的所有目录copy到demo中的CRVideoSDK目录下
* 用Unity Hub新建一个工程
* 将demo目录复制到工程的Assets目录下，点击Unity Editor的Play进行编译
